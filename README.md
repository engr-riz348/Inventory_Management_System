# Inventory Management System (IMS)

This is a feature-rich Inventory Management System (IMS) built with C# using Object-Oriented Programming (OOP) principles. The system features a modern, 3D-styled user interface for an immersive user experience.

## Features

- **Product Management**: Add, search, display, update, and delete products.
- **Category Management**: Differentiated product categories like `Electronics` and `Groceries` with specific attributes.
- **Supplier Management**: Create and assign suppliers to products.
- **Alerts**: Get notified for low stock and expiring groceries.
- **3D User Interface**: A visually appealing and interactive 3D UI built with WinForms.

## Technologies Used

- **C#**: Core backend logic.
- **.NET Framework**: For building the application.
- **WinForms**: For the 3D user interface.
